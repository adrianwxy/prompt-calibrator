## What It Does

Fill in a few fields. Get a structured, copy-ready prompt in real time.

The tool walks you through four core inputs:

- **Task** â€” what you want to accomplish
- **AI Role** â€” who the AI should act as
- **Client / Audience** â€” industry, company size, and situation context
- **Additional Context** â€” audience type, deadline, deliverable stage

Optional style preferences let you control tone (concise, formal, conversational) and format (bullet points, prose, step-by-step).

Every generated prompt automatically includes a grounding rule that instructs the AI to acknowledge assumptions, highlight limitations, and never fabricate insights beyond what the data supports.
